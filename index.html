EXPEN-TRACK


  • This web-app deals with the bill/payment management.
  • Whenever , you go on a trip , you tend to pay for your friends and collect the money later on.
  • This web-app provides you a mechanism to take care of your bills in a very easy manner with a great UI support. You can create a group of your respective friends and split and settle bills accordingly.
  • On the contrary, you can also manage your daily bills too. We all keep a specific limit for one day's total expense. This web-app lets you do so . You can keep a track of the bills and even create a report monthly .
  • I made this web-app during my 4th semester in college. It was made between February,2015 - March,2015 and was a part of college project.
  • This web-app is not deployed on any server ,so there is no live demonstration . The source code is available at my github account . It is built in PHP and Material-UI .

STONEBAKED PIZZA WEBSITE

  • This is my first commercial website project.
  • The website is made for a Bhavnagar ,Gujarat based Pizza Food Chain to increase his business by providing an online interface to book pizza.
  • The website is just like any other e-commerce website but one feature different from every other restaurants is there is a mechanism of CREDITS Collection on every order.
  • This means that on every order ,the customer gets certain credit points which he can use later on to pay in form in virtual money.
  • Right now ,it has only COD option for ordering a pizza due to the owner's decision,but in future ,online payment gateway will be added as and when the owner wants it.
  • This website has a great Admin panel to ease the monitoring process and various other mechanism for the owner to handle the website and provides a great UI for the users designed using Bootstrap and developed in PHP.
  • Well , I made this website in a span of 15 days during the break after my 5th sem final exams in December 2016.
    I haven't listed the screenshots for the website. You can go and checkout the UI and even order one for yourself (Condition: You are in Ahmedabad,Gujarat.)

CSI CODING PORTAL


  • This is a coding portal made for conducting Coding Competitions at college level.
  • It was initially developed by two of my colleagues Shaunak Thakker and Harsh Sodiwala.
  • My part was to update the portal and make it more dynamic and compatible to organize various competitions in a simplified manner.
  • Basically, students can download the input file and pass it as an input to their code and generate an output file which is uploaded here. Then , on successful submission ,points are added accordingly. The admin panel lets us have a simplified support for organizing new competitions on a regular basis.
  • I updated this portal in June,2015.It is built in PHP and the user-end is made using Bootstrap and the admin panel in Material-UI.
  • This portal is deployed on the college server and is up and running successfully . This portal is a part of a committee named Computer Society of India-DDU Student Branch which helds these competitions in the college.
